# crop_animefaces

## usage

        $ animefaces_image.py OUTPUT_IMAGE_(CREATE)_FOLDERNAME
        
        $ animefaces_video.py VIDEO_PATH OUTPUT_(CREATE)_FOLDERNAME

        $ cd ~/.pyenv/versions/VERSION_PATH/share/OpenCV/lbpcascades/
